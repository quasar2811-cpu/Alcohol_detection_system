# Alcohol Detection System Using MQ-3 Sensor

This project demonstrates an alcohol detection system using an Arduino Uno and an MQ-3 alcohol sensor.

## Components
- Arduino Uno
- MQ-3 Sensor
- LEDs
- Resistors
- Breadboard
- Buzzer

## Software Used
- Arduino IDE
- Tinkercad
- EasyEDA Pro

## Files Included
- Project Report
- Arduino Code
- PCB Design
- Gerber Files
